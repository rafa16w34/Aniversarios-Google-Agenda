# Aniversários Google Agenda
Criado por Rafael Alves Faria.

Um programa em python que lê uma planilha formatada e cria eventos de aniversário no Google Agenda, usando a data de nascimento e o nome da pessoa. 

## Como usar?
  

## Limitações :
  - Precisa aceitar os termos toda vez;
  - Cria eventos duplicados se utilizar a mesma planilha;
  - É necessário sempre mudar a planilha da pasta manualmente;
  - Funciona pelo terminal;

## Implementações futuras:
  - Indentificar que um aniversário já foi criado, para evitar duplicações;
  - Ter que fazer login e aceitar os termos somente uma vez;
  - Planilha do programa atualiza automaticamente, sem necessidade de baixar e alterar na pasta;
  - Interface gráfica intuitiva;
